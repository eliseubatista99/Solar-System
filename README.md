# Solar-System
OpenGl Solar System

## Installation
The tool used to wrote this code was Visual Studio.

### Install OpenGL
1. Install GLFW
2. Install GLEW
3. Add header search paths. (On Mac, the configuration is in project/build settings/search paths)
4. Link Libraries libglfw, libglew, libassimp, and OpenGL lib.
5. Run main.cpp
